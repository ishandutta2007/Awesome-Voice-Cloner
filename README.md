# Awesome-Voice-Cloner
## AI Voice Cloning Ecosystem

**Curated List of SaaS Products & Open-Source GitHub Projects**  
*Focused on Personal Voice Cloning*  
**Last updated: March 2026**

This repository tracks notable **SaaS platforms** and **open-source projects** building **AI voice cloning tools** — systems that clone **your personal voice** (or any individual voice) from just a few audio samples (3–60 seconds) for text-to-speech, dubbing, singing, real-time conversion, audiobooks, podcasts, voice agents, and more.

**Emphasis** is strictly on **personal/individual voice cloning** (zero-shot or few-shot from *your own* recordings, high-fidelity replicas of yourself or clients) rather than celebrity voice libraries or pre-built packs.

**Examples** include ElevenLabs, Play.ht, and Resemble.ai (the SaaS leaders) alongside open-source standouts like OpenVoice, RVC, and XTTS-v2.

Tools listed here emphasize **personal voice cloning capabilities** (instant cloning, emotion/style/accent control, multilingual support, real-time performance) rather than generic TTS.

Contributions welcome! Open a PR to add/update entries. Keep descriptions factual and link to official sites.

## Open-Source GitHub Projects

### Dedicated Personal Voice Cloning Projects

| Project Name | Backing Org / Accelerator | Description |
| :--- | :--- | :--- |
| **[OpenVoice](https://github.com/myshell-ai/OpenVoice)** | **Binance Labs** (MyShell) | Instant zero-shot voice cloning with granular control over emotion, accent, and style. |
| **[Coqui TTS (XTTS-v2)](https://github.com/coqui-ai/TTS)** | None (Mozilla Spin-off) | Advanced TTS toolkit; cloning from 6s of audio across 17+ languages. Community-maintained. |
| **[RVC WebUI](https://github.com/RVC-Project/Retrieval-based-Voice-Conversion-WebUI)** | Community | Most popular real-time voice conversion. Huge ecosystem for singing and live voice changing. |

| **[Seed-VC](https://github.com/Plachtaa/seed-vc)** | Community | Zero-shot real-time voice and singing conversion (~300ms latency). Ultra-low-data fine-tuning. |
| **[CosyVoice](https://github.com/FunAudioLLM/CosyVoice)** | Alibaba | Multilingual model with zero-shot cross-lingual cloning and instruction-following. |

**Additional strong options**: Bark (suno-ai/bark), SpeechCraft (Bark-based cloning), and community forks of VITS/So-VITS-SVC.

## Disclaimer

- This is a **community-curated** list — not exhaustive and not an endorsement.
- Always obtain explicit consent before cloning any voice. Respect privacy, licensing, and platform terms.
- AI voice cloning is powerful but **requires ethical use** — avoid deepfakes, misinformation, or unauthorized impersonation. Many tools include built-in safeguards and watermarks.
